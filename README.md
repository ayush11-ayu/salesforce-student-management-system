# Student Management System (Salesforce Administrator Project)

## Project Overview
A Salesforce application to manage students, teachers, courses, enrollments, and attendance.

## Features
- Custom Objects
- Custom Fields
- Lookup Relationships
- Validation Rules
- Record-Triggered Flow
- Reports
- Dashboard

## Objects
- Student
- Teacher
- Course
- Enrollment
- Attendance

## Skills Used
- Salesforce Administration
- Flow Builder
- Reports & Dashboards
- Validation Rules
- Data Modeling
